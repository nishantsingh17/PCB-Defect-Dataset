# PCB-Defect-Dataset
Problem Statement:
Develop an automated inspection system for bare printed circuit boards (PCBs) using machine vision and deep learning techniques, where a camera-based scanning setup traverses the PCB along the X–Y plane to capture high-resolution surface images, and a trained model processes these images to accurately detect, classify, and localize manufacturing defects such as open circuits, short circuits, spurious copper, missing copper, pinholes, and solder mask irregularities; the system must demonstrate robustness to variations in PCB layouts and imaging conditions, and produce reliable outputs that can support real-world industrial quality control without relying on component-level inspection or assembly-stage AOI assumptions.
